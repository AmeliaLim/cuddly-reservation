# Cafe Reservations

### To get started
```
make empty folder
git init
npm init -y
npm install -s express
touch .gitignore
echo "node_modules" > .gitignore
touch app.js
node app.js
```
*Installing .gitignore template on VS Studio. Selection of Node to be ignored.
